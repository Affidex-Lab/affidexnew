## HOW TO CREATE (STEP-BY-STEP)

### USING CANVA (FREE):

1. **Go to Canva.com** (create free account)
2. Click **"Create a design"** ‚Üí **"Custom size"**
3. Enter: **1584 x 396 px** ‚Üí Click **"Create new design"**
4. In left sidebar, click **"Elements"**
5. Search **"gradient"** ‚Üí Choose dark blue/black gradient
6. Drag gradient to fill entire canvas
7. Click **"Text"** ‚Üí Add text boxes for each line
8. Copy text from prompts above
9. Adjust font sizes, colors, and positioning
10. Search **"geometric pattern"** in Elements for background graphics
11. Set pattern opacity to 5-10%
12. Download as **PNG** (high quality)

### USING FIGMA (FREE):

1. **Go to Figma.com** (create free account)
2. Create new file ‚Üí Press **F** for frame
3. Set frame size: **1584 x 396 px**
4. Create rectangle (R key) ‚Üí Fill with gradient
5. Add text (T key) for each line
6. Adjust colors, fonts, and spacing
7. Export as **PNG** (2x resolution)

## REMOVING THE "HIRING" BADGE (OPEN TO WORK)

### TO REMOVE "OPEN TO WORK" FRAME:

1. Go to your LinkedIn profile

3. Click **"Frames"** at the bottom
4. Select **"None"** or remove the frame
5. Click **"Apply"**

1. Click the pencil icon on your profile photo
2. Click **"Frames"**
3. Select **"None"**
